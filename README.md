# Antibiotic Resistance Gene Analysis

This project is a simple analysis of antibiotic resistance genes using Python.

A small dataset was created containing gene names, bacterial organisms, and the type of antibiotic resistance they cause. Using Python in Google Colab, the dataset was analyzed to count the number of genes associated with each resistance type. The results were then visualized using a bar chart.

Tools used:
Python
Pandas
Matplotlib
Google Colab
